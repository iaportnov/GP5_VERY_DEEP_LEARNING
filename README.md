# GP5_VERY_DEEP_LEARNING

## О проекте

Наша предметная область - рынок недвижимости в Екатеринбурге, спрос на которую остается высоким даже при высоких ставках по ипотеке. Ручной анализ каждого объявления это дорого и неэффективно, риелторы и покупатели тратят огромное кол-во времени на то, чтобы отсмотреть все характеристики и оценить состояние ремонта.

Мы предлагаем решение по автоматизированная система оценки квартир, которая анализирует табличные данные (такие как площадь, этаж, год постройки и тд), оценивает состояние ремонта по фотографиям с помощью CV и прогнозирует рыночную стоимость объекта.
Две квартиры с одинаковой площадью и планировкой могут серьезно отличаться по стоимости только из-за состояния ремонта. Наша система позволяет учесть этот фактор и давать более точные оценки.

## Структура репозитория

- datasets – расположены датасеты, полученные во время работы над групповым проектом, в том числе, части датасетов и утилитарные датасеты.

- parsing – расположены ноутбуки с помощью которых доставались данные: 
    1. gp5_parse – сбор данных о квартирах (цены, все доступные поля с характеристиками, описание, ссылки на фотографии) на ekb.cian.ru
    2. download_photos_finall – скачивание всех фотографий по ссылкам


- itog – файл с EDA по извлеченным данным
