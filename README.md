# GP5_VERY_DEEP_LEARNING


Структура репозитория

datasets – расположены датасеты, полученные во время работы над групповым проектом, в том числе, части датасетов и утилитарные датасеты.

parsing – расположены ноутбуки с помощью которых доставались данные: 
    1. gp5_parse – сбор данных о квартирах (цены, все доступные поля с характеристиками, описание, ссылки на фотографии) на ekb.cian.ru
    2. download_photos_finall – скачивание всех фотографий по ссылкам


itog – файл с EDA по извлеченным данным
